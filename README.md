# Yang Jiahao Pages

静态学习资料导航站。目标域名：`website.yangjiahao.top`。

## 目录结构

- `index.html`：导航首页。
- `algebra-book-2-chapter-8/index.html`：代数第2册第八章学习版。
- `algebra-book-2-chapter-8/answers.html`：对应习题答案详解。
- `CNAME`：GitHub Pages 自定义域名配置。

## 新增页面

1. 新建一个子目录，例如 `new-topic/`。
2. 把页面放为 `new-topic/index.html`。
3. 在根目录 `index.html` 中新增一个卡片入口。
4. 提交并推送到 GitHub Pages 仓库。
