一、章节结构

本章主线是：

  1. 认识一元二次方程：标准形式、一般形式、特殊形式。
  2. 掌握一元二次方程的解法：直接开方法、因式分解法、配方法、公式法。
  3. 用判别式判断实根情况。
  4. 用根与系数关系，也就是韦达定理，处理不解方程的问题。
  5. 把二次方程思想扩展到二次三项式因式分解、二元二次多项式因式分解。
  6. 学会把双二次方程、可换元方程、分式方程、无理方程转化为一元二次方程。
  7. 用一元二次方程解决面积、工程、行程、增长率等应用题。

二、核心知识点

1. 一元二次方程

只含有一个未知数，并且未知数最高次数是 2 的整式方程，叫一元二次方程。

一般形式：

公式
a·x² + b·x + c = 0, 其中 a ≠ 0

其中 a·x² 是二次项，b·x 是一次项，c 是常数项，a, b, c 分别是二次项系数、一次项系数、常数项。

常见特殊形式：

公式
a·x² + c = 0
a·x² + b·x = 0
a·x² = c

补充要点：

  • 判断是不是一元二次方程前，先把方程整理成整式方程。
  • 二次项系数不能为 0。若含字母参数，必须讨论使二次项系数为 0 的情况。
  • 分式方程、无理方程即使化简后出现二次项，也不能直接说原方程是一元二次方程。

2. 不完全一元二次方程

不完全一元二次方程是 b=0 或 c=0 的一元二次方程。

a·x² + c = 0：

公式
x² = -c/a

若 -c/a > 0，有两个实根；若 -c/a = 0，有两个相等实根 0；若 -c/a < 0，没有实数根。

a·x² + b·x = 0：

公式
x(ax + b) = 0
x = 0 或 x = -b/a

易错点：

  • 从 x² = k 得 x = ±√(k)，不能只取正根。
  • x(ax+b)=0 用的是“两个因式积为 0，则至少一个因式为 0”。

3. 因式分解法

如果一元二次方程可以化为：

公式
A(x)B(x) = 0

则：

公式
A(x)=0 或 B(x)=0

常用分解方式：提公因式、平方差、完全平方、十字相乘。

适用场景：方程容易分解因式时，因式分解法最快。

4. 配方法

配方法的目标是把方程变为：

公式
(x + m)² = n

标准步骤，以 a·x²+b·x+c=0 为例：

  1. 二次项系数化为 1。
  2. 常数项移到右边。
  3. 两边同时加上一次项系数一半的平方。
  4. 左边写成完全平方，再开平方。

对 x² + px + q = 0：

公式
x² + px = -q
x² + px + (p/2)² = (p/2)² - q
(x + p/2)² = p²/4 - q

补充：配方法不仅是解法，也是推导求根公式和研究判别式的基础。

5. 公式法

对一元二次方程：

公式
a·x² + b·x + c = 0, a ≠ 0

求根公式：

公式
x = (-b ± √(b² - 4ac)) / (2a)

使用步骤：

  1. 整理成一般形式。
  2. 明确 a, b, c。
  3. 计算 Δ = b² - 4ac。
  4. 若 Δ ≥ 0，代入公式求实根；若 Δ < 0，在实数范围内无根。

易错点：

  • a 是二次项系数，不一定是 1。
  • b 带符号，例如 3x² - 5x + 2 = 0 中 b=-5。
  • 分母是 2a，不是 2。

6. 根的判别式

公式
Δ = b² - 4ac

对 a·x²+b·x+c=0 (a ≠ 0)：

  • Δ > 0：有两个不相等的实数根。
  • Δ = 0：有两个相等的实数根。
  • Δ < 0：没有实数根。

补充：判别式只判断根的实数情况，不直接给出根。需要根时仍要用公式法、因式分解法等。

7. 列一元二次方程解应用题

基本流程：

  1. 审题，找等量关系。
  2. 设未知数，注意单位。
  3. 根据等量关系列方程。
  4. 解方程。
  5. 检验根是否符合实际意义。
  6. 作答。

常见模型：

  • 面积问题：长、宽、面积之间的关系。
  • 行程问题：路程 = 速度 × 时间。
  • 工程问题：工作总量 = 工作效率 × 时间。
  • 增长率问题：原量 × (1+r)^n = 后量。
  • 几何问题：勾股定理、周长、面积。

8. 韦达定理：根与系数关系

若 x1, x2 是 a·x²+b·x+c=0 (a ≠ 0) 的两个根，则：

公式
x1 + x2 = -b/a
x1 x2 = c/a

特别地，对 x² + px + q = 0：

公式
x1 + x2 = -p
x1 x2 = q

常用变形：

公式
x1² + x2² = (x1+x2)² - 2x1x2
1/x1 + 1/x2 = (x1+x2)/(x1x2), 前提 x1x2 ≠ 0
(x1-x2)² = (x1+x2)² - 4x1x2

由根作方程：若两根是 m, n，则可取：

公式
x² - (m+n)x + mn = 0

若要求首项系数为 a，则：

公式
a[x² - (m+n)x + mn] = 0

9. 二次三项式的根与因式分解

二次三项式：

公式
a·x² + b·x + c, a ≠ 0

它的根就是方程 a·x²+b·x+c=0 的根。

若根为 x1, x2，则：

公式
a·x² + b·x + c = a(x - x1)(x - x2)

若 a=1：

公式
x² + px + q = (x - x1)(x - x2)

易错点：当二次项系数不是 1 时，不能漏掉前面的 a。

10. 十字相乘法

十字相乘法用于分解形如：

公式
a·x² + b·x + c

思路：把 a 分成两个因数的积，把 c 分成两个因数的积，交叉相乘后相加得到 b。

例：

公式
6x² - 11x - 10
= (2x - 5)(3x + 2)

因为交叉项为 4x - 15x = -11x。

11. 二元二次多项式的因式分解

如：

公式
2x² + xy - 3y²

可以把它看作关于 x 的二次三项式，y 作为系数或常数的一部分：

公式
2x² + xy - 3y² = (x - y)(2x + 3y)

也可以把它看作关于 y 的二次三项式。两种做法应得到相同结果。

12. 双二次方程

只含未知数偶次项的一元四次方程叫双二次方程，常见形式：

公式
ax⁴ + b·x² + c = 0, a ≠ 0

令：

公式
y = x²

则化为：

公式
ay² + by + c = 0

求出 y 后，再解 x² = y。

注意：若 y < 0，在实数范围内没有对应的 x。

13. 换元法

有些方程虽然不是直接的双二次方程，但含有重复结构，可以设辅助未知数。

例：

公式
(x² + 2x)² - 14(x² + 2x) - 15 = 0

令 y = x² + 2x，得到：

公式
y² - 14y - 15 = 0

解出 y 后再回代。

使用换元法时要注意：

  • 设元要覆盖重复出现的整体。
  • 求出辅助未知数后必须回到原未知数。
  • 可能出现无实数解或增根，需要检验。

14. 分式方程

含有未知数的分式方程，通常先去分母，化为整式方程。若化为一元二次方程，可用本章方法求解。

关键步骤：

  1. 确定最简公分母。
  2. 方程两边同乘最简公分母，去分母。
  3. 解所得整式方程。
  4. 检验：使原方程分母为 0 的根是增根，必须舍去。

15. 无理方程

根号内含有未知数的方程叫无理方程。

常用方法：移项、平方、再整理成有理方程。

关键步骤：

  1. 先隔离一个根式，使平方后尽量简单。
  2. 两边平方，化为有理方程。
  3. 解所得方程。
  4. 代回原方程检验。

为什么必须验根：平方变形可能扩大解集，例如 A=B 平方后得到 A²=B²，会同时包含 A=B 和 A=-B 的情况。

三、学习路线建议

  1. 先熟练整理一般形式 a·x²+b·x+c=0。
  2. 再按优先级选择解法：
公式
能直接开方 -> 直接开方法
能快速分解 -> 因式分解法
需要推导或题目要求 -> 配方法
一般情况 -> 公式法
只判断根情况 -> 判别式
不要求解根但要求根的和积 -> 韦达定理
  1. 学会转化思想：很多后续题目不是新方法，而是先把题目化成一元二次方程。
  2. 分式方程和无理方程必须验根。

I. 应用题

  1. 一个矩形的面积是 40 平方厘米，长比宽多 6 厘米。求长和宽。

六、易错点清单

  1. 没有先整理成 a·x²+b·x+c=0 就套公式。
  2. 忘记二次项系数 a 不能为 0。
  3. x²=k 只写一个根。
  4. 因式分解后只令一个因式为 0。
  5. 二次项系数不是 1 时，二次三项式分解漏掉首项系数。
  6. 用判别式时把 b 的符号写错。
  7. 韦达定理中 x1+x2=-b/a 的负号漏掉。
  8. 分式方程去分母后不验增根。
  9. 无理方程平方后不验根。
  10. 应用题得到两个根后，没有根据实际意义舍去不合理根。
