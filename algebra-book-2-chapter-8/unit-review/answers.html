第八章单元综合复习练习题答案详解

1. 方程整理与求解

已知：(x+2)(x-3)=2x-6。

公式
(x+2)(x-3)=x²-x-6
x²-x-6=2x-6
x²-3x=0
公式
所以一般形式为：
x²-3x=0

它是关于 x 的一元二次方程，因为只含一个未知数，最高次数是 2，且二次项系数为 1。

公式
解方程：
x²-3x=0
x(x-3)=0

答案：x=0 或 x=3。

2. 解法选择与验算

方程为：2x² - 7x + 3 = 0。

这个方程可以因式分解，优先用因式分解法。

公式
2x² - 7x + 3 = (2x-1)(x-3)
(2x-1)(x-3)=0
公式
所以：
2x-1=0 或 x-3=0

答案：x=1/2 或 x=3。

公式
验算 x=3：
2·3² - 7·3 + 3 = 18 - 21 + 3 = 0

所以 x=3 是原方程的根。

3. 判别式与根的情况

方程为：x² - 6x + 9 = 0。

公式
a=1, b=-6, c=9
Δ=b²-4ac=(-6)²-4·1·9=36-36=0

判别式等于 0，所以方程有两个相等的实数根。

公式
解方程：
x² - 6x + 9 = (x-3)²
(x-3)²=0

答案：x=3，且是重根。结果与判别式判断一致。

4. 不完全方程与因式分解

方程为：3x² - 12x = 0。

这是 a·x²+b·x=0 型的不完全一元二次方程。

公式
3x² - 12x = 0
3x(x-4)=0
公式
所以：
x=0 或 x=4
公式
两根的和：
0+4=4
公式
两根的积：
0·4=0

5. 二次三项式与方程根

二次三项式为：x² - 5x + 6。

公式
令它等于 0：
x² - 5x + 6 = 0
(x-2)(x-3)=0

所以二次三项式的根为 2 和 3。

公式
分解因式：
x² - 5x + 6 = (x-2)(x-3)

二次三项式的根，就是使该二次三项式等于 0 的 x 的值，也就是对应方程 x²-5x+6=0 的根。

6. 公式法与近似思想

方程为：x² - 3x - 1 = 0。

公式
a=1, b=-3, c=-1
Δ=(-3)²-4·1·(-1)=13
公式
公式法：
x = (3 ± √(13))/2
公式
因为：
3 < √(13) < 4
公式
所以：
(3+√(13))/2 在 3 和 4 之间
(3-√(13))/2 在 -1 和 0 之间

答案：x=(3+√(13))/2 或 x=(3-√(13))/2。

7. 韦达定理与直接求解对照

方程为：x² - 7x + 10 = 0。

公式
由韦达定理：
x1+x2=7
x1x2=10
公式
解方程：
x² - 7x + 10 = (x-2)(x-5)

答案：x=2 或 x=5。

公式
验证：
2+5=7
2·5=10

符合韦达定理。

8. 几何建模

设矩形宽为 x 厘米，则长为 x+7 厘米。

公式
根据面积为 60：
x(x+7)=60
x²+7x-60=0
公式
分解因式：
(x+12)(x-5)=0
公式
所以：
x=5 或 x=-12

宽不能为负，所以取 x=5。

答案：宽 5 厘米，长 12 厘米。

9. 由实际问题判断根

设这个数为 x。

公式
“平方比这个数的 5 倍多 6”表示：
x² = 5x + 6
x² - 5x - 6 = 0
公式
分解因式：
(x-6)(x+1)=0
公式
所以：
x=6 或 x=-1

如果要求这个数是正数，应取 x=6。

10. 根与方程构造

两个根是 2 和 5。

公式
两根和：
2+5=7
公式
两根积：
2·5=10
公式
二次项系数为 1 的方程：
x² - 7x + 10 = 0
公式
二次项系数为 3 的方程：
3(x² - 7x + 10)=0
3x² - 21x + 30=0

两个方程的两根和都是 7，两根积都是 10。

11. 配方法、公式法与判别式

方程为：x² - 4x - 2 = 0。

公式
配方法：
x² - 4x = 2
x² - 4x + 4 = 6
(x-2)² = 6
x-2 = ±√(6)
公式
所以：
x = 2 ± √(6)
公式
公式法：
a=1, b=-4, c=-2
Δ=(-4)² - 4·1·(-2)=16+8=24
x=(4 ± √(24))/2=2 ± √(6)

两种方法结果一致。

判别式 Δ=24>0，所以方程有两个不相等的实数根。

12. 含参数方程的根的情况

方程为：x² - 2kx + k + 2 = 0。

公式
a=1, b=-2k, c=k+2
Δ=(-2k)² - 4(k+2)
= 4k² - 4k - 8
= 4(k²-k-2)
= 4(k-2)(k+1)
公式
(1) 有两个不相等实数根：
Δ>0
(k-2)(k+1)>0

答案：k>2 或 k<-1。

公式
(2) 有两个相等实数根：
Δ=0
(k-2)(k+1)=0

答案：k=2 或 k=-1。

公式
(3) 当 k=3 时：
x² - 6x + 5 = 0
(x-1)(x-5)=0

答案：x=1 或 x=5。

13. 韦达定理综合计算

方程为：2x² - 5x - 3 = 0。

公式
由韦达定理：
x1+x2 = -b/a = 5/2
x1x2 = c/a = -3/2
公式
平方和：
x1²+x2²=(x1+x2)²-2x1x2
=(5/2)² - 2·(-3/2)
=25/4+3
=37/4
公式
倒数和：
1/x1 + 1/x2 = (x1+x2)/(x1x2)
= (5/2)/(-3/2)
= -5/3
公式
直接解方程：
2x² - 5x - 3 = (2x+1)(x-3)
公式
所以 x= -1/2 或 x=3。验证：
-1/2 + 3 = 5/2
(-1/2)·3 = -3/2

14. 二次三项式因式分解与方程求解

公式
对应方程：
2x² - 7x + 3 = 0
公式
分解：
2x² - 7x + 3 = (2x-1)(x-3)
公式
所以根为：
x=1/2 或 x=3
公式
利用根分解：
2x² - 7x + 3 = 2(x-1/2)(x-3)
公式
化简后与十字相乘结果一致：
2(x-1/2)(x-3) = (2x-1)(x-3)

15. 双二次方程与实数根判断

方程为：x⁴ - 5x² + 4 = 0。

公式
因为只含 x 的偶次项，所以可以令：
y=x²
公式
原方程化为：
y² - 5y + 4 = 0
(y-1)(y-4)=0
公式
所以：
y=1 或 y=4
公式
回代：
x²=1 -> x=±1
x²=4 -> x=±2

答案：x=±1, ±2。

16. 分式方程与增根检验

方程为：1/(x-1) + 2/(x+1) = 1。

公式
取值限制：
x ≠ 1, x ≠ -1
公式
两边同乘 (x-1)(x+1)：
x+1 + 2(x-1) = x² - 1
x+1+2x-2 = x² - 1
3x-1 = x² - 1
x² - 3x = 0
x(x-3)=0
公式
所以：
x=0 或 x=3

检验：0 和 3 都不等于 1 或 -1，所以都不是增根。

答案：x=0 或 x=3。

17. 工程问题建模

设甲单独完成需要 x 天，则乙单独完成需要 x+10 天。

公式
甲、乙合作 12 天完成，所以：
1/x + 1/(x+10) = 1/12
公式
两边同乘 12x(x+10)：
12(x+10)+12x=x(x+10)
24x+120=x²+10x
x²-14x-120=0
公式
分解因式：
(x-20)(x+6)=0
公式
所以：
x=20 或 x=-6

时间不能为负，取 x=20。

答案：甲单独完成需要 20 天，乙单独完成需要 30 天。

18. 无理方程与验根

方程为：√(x+4)=x-2。

公式
限制条件：
x+4≥0
x-2≥0

所以候选根必须满足 x≥2。

公式
两边平方：
x+4=(x-2)²
x+4=x²-4x+4
x²-5x=0
x(x-5)=0

候选根：x=0 或 x=5。

公式
检验：
x=0 不满足 x≥2，舍去
x=5: √(9)=3, 5-2=3，成立

答案：x=5。

19. 参数、韦达定理与判别式综合

方程为：x² - (m+3)x + 2m + 2 = 0。

公式
由韦达定理：
x1+x2=m+3
x1x2=2m+2
公式
又因为：
x1²+x2²=(x1+x2)²-2x1x2=13
公式
代入：
(m+3)² - 2(2m+2)=13
m²+6m+9-4m-4=13
m²+2m-8=0
(m+4)(m-2)=0
公式
所以：
m=-4 或 m=2

检查实数根：

公式
当 m=-4：
方程为 x²+x-6=0
Δ=1+24=25>0
公式
当 m=2：
方程为 x²-5x+6=0
Δ=25-24=1>0

两个值都使原方程有实数根。

答案：m=-4 或 m=2。

20. 综合应用题：行程问题

设去时速度为 x 千米/小时，则返回速度为 x-10 千米/小时。

公式
根据“返回所用时间比去时多 1 小时”：
120/(x-10) - 120/x = 1
公式
去分母：
120x - 120(x-10) = x(x-10)
1200 = x² - 10x
x² - 10x - 1200 = 0
公式
解方程：
(x-40)(x+30)=0
公式
所以：
x=40 或 x=-30

速度不能为负，且返回速度 x-10 必须为正，所以取 x=40。

答案：去时速度为 40 千米/小时，返回速度为 30 千米/小时。
