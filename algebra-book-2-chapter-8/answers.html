《代数 第2册》第八章精选练习题答案详解

1. 判断是否为一元二次方程

(1) 3x² - 2x + 1 = 0

这是关于 x 的整式方程，只含一个未知数，最高次数是 2，并且二次项系数 3 ≠ 0，所以是一元二次方程。

(2) x + 1/x = 2

原方程含有 1/x，未知数在分母中，不是整式方程，所以它本身不是一元二次方程。若两边同乘 x，会得到 x² - 2x + 1 = 0，但这是变形后的整式方程，不能改变原方程的类型判断。

(3) (x+1)² = x² + 3

先化简：

公式
(x+1)² = x² + 2x + 1
x² + 2x + 1 = x² + 3
2x + 1 = 3

化简后最高次数是 1，所以不是一元二次方程。

(4) (a-1)x² + 2x - 3 = 0，未知数为 x

当 a-1 ≠ 0，即 a ≠ 1 时，二次项系数不为 0，是一元二次方程；当 a=1 时，方程变成 2x-3=0，不是一元二次方程。

2. 解方程

(1) x² - 25 = 0

公式
x² = 25
x = ±5

答案：x=5 或 x=-5。

(2) 3x² - 12 = 0

公式
3x² = 12
x² = 4
x = ±2

答案：x=2 或 x=-2。

(3) 2x² - 5x = 0

公式
x(2x-5)=0
x=0 或 2x-5=0

答案：x=0 或 x=5/2。

(4) (x-3)² = 16

公式
x-3 = ±4

所以：

公式
x=7 或 x=-1

3. 用因式分解法解方程

(1) x² - 5x + 6 = 0

公式
x² - 5x + 6 = (x-2)(x-3)
(x-2)(x-3)=0

答案：x=2 或 x=3。

(2) 2x² + 5x - 3 = 0

公式
2x² + 5x - 3 = (2x-1)(x+3)
(2x-1)(x+3)=0

答案：x=1/2 或 x=-3。

(3) 6x² - 11x - 10 = 0

公式
6x² - 11x - 10 = (2x-5)(3x+2)
(2x-5)(3x+2)=0

答案：x=5/2 或 x=-2/3。

(4) (x+2)(x-5) = 18

先整理：

公式
x² - 3x - 10 = 18
x² - 3x - 28 = 0
(x-7)(x+4)=0

答案：x=7 或 x=-4。

4. 用配方法解方程

(1) x² + 6x + 5 = 0

公式
x² + 6x = -5
x² + 6x + 9 = 4
(x+3)² = 4
x+3 = ±2

答案：x=-1 或 x=-5。

(2) x² - 4x - 1 = 0

公式
x² - 4x = 1
x² - 4x + 4 = 5
(x-2)² = 5
x-2 = ±√(5)

答案：x=2+√(5) 或 x=2-√(5)。

5. 用公式法解方程

(1) 3x² - 5x - 2 = 0

这里 a=3, b=-5, c=-2。

公式
Δ = b² - 4ac = (-5)² - 4·3·(-2) = 25 + 24 = 49
x = (5 ± √(49)) / 6 = (5 ± 7) / 6

答案：x=2 或 x=-1/3。

(2) 2x² + 2x - 1 = 0

这里 a=2, b=2, c=-1。

公式
Δ = 2² - 4·2·(-1) = 12
x = (-2 ± √(12)) / 4 = (-2 ± 2√(3)) / 4

答案：x=(-1+√(3))/2 或 x=(-1-√(3))/2。

(3) x² + 2x + 3 = 0

公式
Δ = 2² - 4·1·3 = -8 < 0

答案：在实数范围内无解。

6. 判断根的情况

(1) x² - 4x + 4 = 0

公式
Δ = (-4)² - 4·1·4 = 16 - 16 = 0

答案：有两个相等的实数根。

(2) 2x² - 3x - 1 = 0

公式
Δ = (-3)² - 4·2·(-1) = 9 + 8 = 17 > 0

答案：有两个不相等的实数根。

(3) 5x² + 2x + 1 = 0

公式
Δ = 2² - 4·5·1 = 4 - 20 = -16 < 0

答案：没有实数根。

7. 求参数 k

方程为：

公式
x² - 2kx + k + 3 = 0

要有两个不相等的实数根，需要 Δ > 0。

公式
Δ = (-2k)² - 4·1·(k+3)
= 4k² - 4k - 12
= 4(k²-k-3)

所以：

公式
k² - k - 3 > 0

解对应二次方程：

公式
k² - k - 3 = 0
k = (1 ± √(13)) / 2

因为二次项系数为正，不等式大于 0 时取两侧区间。

答案：

公式
k < (1-√(13))/2 或 k > (1+√(13))/2

8. 用韦达定理求值

方程：2x² - 3x - 5 = 0，两根为 x1, x2。

由韦达定理：

公式
x1 + x2 = -b/a = -(-3)/2 = 3/2
x1x2 = c/a = -5/2

(1) 答案：x1+x2=3/2。

(2) 答案：x1x2=-5/2。

(3)

公式
x1² + x2² = (x1+x2)² - 2x1x2
= (3/2)² - 2·(-5/2)
= 9/4 + 5
= 29/4

(4)

公式
1/x1 + 1/x2 = (x1+x2)/(x1x2)
= (3/2)/(-5/2)
= -3/5

9. 由根作方程

两根为：

公式
3+√(2), 3-√(2)

两根和：

公式
(3+√(2)) + (3-√(2)) = 6

两根积：

公式
(3+√(2))(3-√(2)) = 9 - 2 = 7

所以方程可取：

公式
x² - 6x + 7 = 0

10. 分解因式

(1) x² - 7x + 10

找两个数，和为 -7，积为 10，是 -5 和 -2。

公式
x² - 7x + 10 = (x-5)(x-2)

(2) 3x² + 10x + 8

十字相乘：

公式
3x² + 10x + 8 = (3x+4)(x+2)

检验交叉项：6x+4x=10x。

(3) 2x² + xy - 3y²

把它看作关于 x 的二次式，分解为：

公式
2x² + xy - 3y² = (x-y)(2x+3y)

检验：3xy-2xy=xy。

(4) x² - xy - 6y² + x + 7y - 2

尝试分解成两个一次因式。可验证：

公式
(x-3y+2)(x+2y-1)

展开：

公式
x² + 2xy - x - 3xy - 6y² + 3y + 2x + 4y - 2
= x² - xy - 6y² + x + 7y - 2

所以答案是：

公式
(x-3y+2)(x+2y-1)

11. 解双二次方程

(1) x⁴ - 13x² + 36 = 0

令 y=x²，得：

公式
y² - 13y + 36 = 0
(y-4)(y-9)=0

所以 y=4 或 y=9。

公式
x²=4 -> x=±2
x²=9 -> x=±3

答案：x=±2, ±3。

(2) x⁴ + 5x² - 36 = 0

令 y=x²，得：

公式
y² + 5y - 36 = 0
(y+9)(y-4)=0

所以 y=4 或 y=-9。因为 x²=-9 在实数范围内无解，只保留 x²=4。

答案：x=±2。

(3) x⁴ + 5x² + 4 = 0

令 y=x²，得：

公式
y² + 5y + 4 = 0
(y+1)(y+4)=0

所以 y=-1 或 y=-4。二者都小于 0，不能等于实数平方。

答案：在实数范围内无解。

12. 用换元法解方程

(1) (x² + 2x)² - 14(x² + 2x) - 15 = 0

令：

公式
y = x² + 2x

原方程化为：

公式
y² - 14y - 15 = 0
(y-15)(y+1)=0

所以 y=15 或 y=-1。

当 y=15：

公式
x² + 2x = 15
x² + 2x - 15 = 0
(x+5)(x-3)=0

得到 x=-5 或 x=3。

当 y=-1：

公式
x² + 2x = -1
x² + 2x + 1 = 0
(x+1)²=0

得到 x=-1。

答案：x=-5, 3, -1，其中 -1 是重根。

(2) (x² - 3x + 1)² - 11(x² - 3x + 1) + 10 = 0

令：

公式
y = x² - 3x + 1

原方程化为：

公式
y² - 11y + 10 = 0
(y-1)(y-10)=0

所以 y=1 或 y=10。

当 y=1：

公式
x² - 3x + 1 = 1
x² - 3x = 0
x(x-3)=0

得到 x=0 或 x=3。

当 y=10：

公式
x² - 3x + 1 = 10
x² - 3x - 9 = 0

公式法：

公式
Δ = (-3)² - 4·1·(-9) = 45
x = (3 ± √(45))/2 = (3 ± 3√(5))/2

答案：

公式
x=0, x=3, x=(3+3√(5))/2, x=(3-3√(5))/2

13. 解分式方程并检验

(1) 1/(x-1) + 1/(x+1) = 1

分母要求：x ≠ 1 且 x ≠ -1。

两边同乘 (x-1)(x+1)：

公式
x+1 + x-1 = x² - 1
2x = x² - 1
x² - 2x - 1 = 0

公式法：

公式
x = (2 ± √(4+4))/2 = 1 ± √(2)

检验：1+√(2) 和 1-√(2) 都不等于 1 或 -1，所以都不是增根。

答案：x=1+√(2) 或 x=1-√(2)。

(2) 3/(x-2) = x/(x-2) + 2

分母要求：x ≠ 2。

两边同乘 x-2：

公式
3 = x + 2(x-2)
3 = x + 2x - 4
3x = 7
x = 7/3

检验：7/3 ≠ 2，所以不是增根。

答案：x=7/3。

14. 解无理方程并检验

(1) √(x+5) = x - 1

根号要求：x+5 ≥ 0，右边还要满足 x-1 ≥ 0，所以候选解应有 x ≥ 1。

两边平方：

公式
x+5 = (x-1)²
x+5 = x² - 2x + 1
x² - 3x - 4 = 0
(x-4)(x+1)=0

候选根：x=4 或 x=-1。

检验：

公式
x=4: √(9)=3, x-1=3，成立
x=-1: √(4)=2, x-1=-2，不成立

答案：x=4。

(2) √(2x+3) = x

根号要求：2x+3 ≥ 0，且右边 x 必须非负，所以候选解应有 x ≥ 0。

两边平方：

公式
2x+3 = x²
x² - 2x - 3 = 0
(x-3)(x+1)=0

候选根：x=3 或 x=-1。

检验：

公式
x=3: √(9)=3，成立
x=-1: √(1)=1，但右边为 -1，不成立

答案：x=3。

15. 矩形面积应用题

设宽为 x 厘米，则长为 x+6 厘米。

根据面积为 40：

公式
x(x+6)=40
x²+6x-40=0
(x+10)(x-4)=0

得到 x=4 或 x=-10。宽不能为负，所以取 x=4。

答案：宽 4 厘米，长 10 厘米。

16. 生产计划应用题

设原计划每天生产 x 个零件，则总量为 10x。

实际提前 2 天完成，所以实际用了 8 天；实际每天生产 x+20 个。

列方程：

公式
8(x+20)=10x
8x+160=10x
2x=160
x=80

答案：原计划每天生产 80 个零件。

17. 一个数与倒数之和

设这个数为 x。因为有倒数，所以 x ≠ 0。

公式
x + 1/x = 5/2

两边同乘 2x：

公式
2x² + 2 = 5x
2x² - 5x + 2 = 0
(2x-1)(x-2)=0

答案：x=1/2 或 x=2。

18. 连续正整数

设两个连续正整数为 n 和 n+1。

公式
n(n+1)=156
n²+n-156=0

分解因式：

公式
n²+n-156 = (n-12)(n+13)

所以 n=12 或 n=-13。因为题目要求正整数，舍去 -13。

答案：这两个数是 12 和 13。
